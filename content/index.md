# Fd-server

- template: page
- pubdate: 2014-04-18
- description: Fds 是一套统一的本地开发环境，面向工程化的前端开发项目。简化开发人员生产时对服务配置的依赖，提高前端开发人员的开发效率。面向自动化的项目流程管理，交付，测试。减少复杂的人工操作。自身包含的扩展机制，也能够提供给开发者灵活多变的个性化服务定制方案,同时Fds也是一套跨平台的本地开发环境，完美支持win,linux,mac os。
----------
